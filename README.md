# TuSDK faceAPI PHP Demo
***
支持 PHP5.3+，依赖扩展[curl](http://www.php.net/curl)


## 1. 开通服务
* 您需要先 [注册](http://tusdk.com/center/user/register) 成为 TuSDK 用户
* 联系商务并开通人脸API服务
* 在控制台 [云API](http://tusdk.com/center/apiService) 页面获取您的 `API私有key` 和 `API公有key`

## 2. 使用
* 打开文件 TuFace.php, 把类 `TuFace` 的 `key` 属性修改为您的 `API私有key`，把类 `TuFace` 的 `pid` 属性修改为您的 `API公有key`
* 打开文件 demo.php, 修改 `$file` 变量为您需要发送的图片地址或URL
* 运行示例文件 demo.php

更多信息请访问 [tusdk.com](http://tusdk.com)。
